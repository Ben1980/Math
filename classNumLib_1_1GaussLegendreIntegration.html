Detailed Description

A numerical integration method using the Gauss-Legendre approach.

GaussLegendreIntegration is using the Gauss-Legendre approach to acomplish numerical integrations as described at thoughts-on-coding.com and calculates the integral of function \(f\) in range \([a-b]\).
